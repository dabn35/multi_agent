# S.E.E. v2 - Skin & Eating Habits Experience

## 프로젝트 목표

식습관과 피부 상태의 상관관계를 추적하고, 개인의 피부 상태에 맞춘 화장품 성분과 루틴을 추천하는 Streamlit 기반 에이전트

## 핵심 기능

1. **날짜별 식단 기록**
   - 아침, 점심, 저녁, 간식 4개 섹션
   - 7가지 식사 카테고리 + 6가지 간식 카테고리

2. **피부 상태 추적**
   - 5가지 계열(수분/유분/민감/트러블/기타), 14가지 세부 항목
   - 토글형 버튼 UI로 다중 선택 가능

3. **통계 및 분석**
   - 최근 7일간 음식 섭취 패턴 분석
   - 피부 증상 발생 빈도 통계
   - 자동 생성 요약 리포트

4. **규칙 기반 화장품 추천**
   - 선택된 피부 상태에 맞는 성분/루틴 추천
   - 건조/민감/트러블 케어 세부 가이드

## 데이터 저장

- JSON 형식으로 누적 저장 (`data/daily_records.json`)
- Streamlit session_state로 메모리 관리

## 실행 방법

```bash
streamlit run see_v2.py
```
